# 10. Кейсы из Реальной Практики

В этой главе мы рассмотрим примеры агентов в разных доменах.

## DevOps Agent

**Задача:** "У нас проблемы с базой, разберись"

**Действия агента:**
1. Проверяет метрики БД
2. Читает логи
3. Выдвигает гипотезы
4. Применяет фиксы
5. Верифицирует результат

## Customer Support Agent

**Задача:** "Пользователь жалуется на медленную загрузку"

**Действия агента:**
1. Получает тикет
2. Ищет в базе знаний
3. Собирает контекст (версия браузера, ОС)
4. Формулирует ответ
5. Эскалирует при необходимости

## Data Analytics Agent

**Задача:** "Почему упали продажи в регионе X?"

**Действия агента:**
1. Формулирует SQL-запрос
2. Проверяет качество данных
3. Анализирует тренды
4. Генерирует отчет

## Security Agent (SOC)

**Задача:** "Алерт: подозрительная активность на хосте 192.168.1.10"

**Действия агента:**
1. Триажирует алерт
2. Собирает доказательства (логи, метрики)
3. Определяет severity
4. Изолирует хост (с подтверждением)
5. Генерирует отчет

## Product Operations Agent

**Задача:** "Подготовь план релиза фичи X"

**Действия агента:**
1. Собирает требования
2. Проверяет зависимости
3. Создает документы
4. Отправляет на согласование
